# Web

## Назначение
Фронтенд‑часть веб‑сайта для заказа еды через браузер.

## Компоненты
* главная страница и каталог ресторанов;
* меню и карточки блюд;
* корзина и форма оформления заказа;
* личный кабинет (история заказов, профиль);
* админ‑панель для ресторанов (управление меню, заказами).

## Взаимодействие
Веб‑сайт получает данные (меню, статус заказа) и отправляет запросы через `ServerApi`, используя тот же API, что и мобильное приложение.
