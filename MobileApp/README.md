# MobileApp

## Назначение
Код мобильного приложения для iOS и Android (для клиентов и курьеров).

## Компоненты
* экраны каталога и меню ресторанов;
* корзина и оформление заказа;
* отслеживание статуса заказа в реальном времени;
* профили пользователей и настройки;
* система уведомлений;
* интерфейс для курьеров (приём заказов, отметка статусов доставки).

## Взаимодействие
Приложение получает данные (меню, рестораны, заказы) и отправляет запросы через `ServerApi`.
