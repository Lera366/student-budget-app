# ServerApi

## Назначение
Бэкенд‑часть приложения — API для обработки данных и бизнес‑логики.

## Компоненты
* API‑эндпоинты для мобильных и веб‑клиентов;
* бизнес‑логика (обработка заказов, расчёты, скидки);
* аутентификация и авторизация пользователей;
* управление ресторанами, меню, заказами;
* интеграция с платёжными системами;
* уведомления (SMS, push).

## Взаимодействие
API служит центральным звеном:
* принимает запросы от `MobileApp` и `Web`;
* передаёт данные клиентам;
* взаимодействует с БД и внешними сервисами (платежи, карты).
